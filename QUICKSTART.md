# Coffee Track - Быстрый старт

## Запуск проекта за 3 шага

### 1. Установите зависимости
```bash
npm install
```

### 2. Запустите сервер разработки
```bash
npm run dev
```

### 3. Откройте в браузере
```
http://localhost:5173
```

## Основные команды

```bash
# Разработка
npm run dev          # Запустить dev сервер

# Продакшен
npm run build        # Собрать проект
npm run preview      # Предпросмотр собранной версии

# Линтинг
npm run lint         # Проверить код
```

## Как использовать

1. **Добавить напиток**: Нажмите "+ Добавить напиток" → заполните форму → "Добавить"
2. **Просмотреть детали**: Кликните на карточку напитка
3. **Редактировать**: Откройте детали → "Редактировать"
4. **Удалить**: Нажмите ✕ на карточке
