# Coffee Track

Веб-приложение для отслеживания приготовления кофейных напитков. Записывайте параметры приготовления, оценивайте результат и ведите историю всех ваших кофейных экспериментов!

## Возможности

- **Добавление напитков**: записывайте детальную информацию о каждом приготовленном напитке
  - Тип напитка (Гейзерная, French Press)
  - Сорт/бренд кофе
  - Степень помола (Mahlgrad)
  - Количество кофе (граммы)
  - Количество воды (мл)
  - Количество молока (мл, опционально)
  - Время приготовления (опционально)
  - Особенности приготовления (заметки)
  - Оценка от 1 до 5 звезд
  - Текстовый отзыв с подробными заметками о вкусе (опционально)

- **Просмотр истории**: все ваши напитки в удобном списке с сортировкой по дате
- **Детальная информация**: просматривайте полную информацию о каждом напитке
- **Редактирование**: изменяйте записи о напитках
- **Удаление**: удаляйте ненужные записи
- **Локальное хранение**: все данные сохраняются в браузере, не требуется интернет
- **Облачная синхронизация**: синхронизируйте данные между устройствами через Supabase (опционально)
- **Мобильная адаптация**: оптимизировано для использования на смартфонах

## Технологии

- **React 18** - библиотека для создания пользовательского интерфейса
- **TypeScript** - статическая типизация
- **Vite** - быстрая сборка и разработка
- **LocalStorage API** - хранение данных в браузере
- **Supabase** - облачная база данных для синхронизации (опционально)
- **CSS3** - адаптивный дизайн

## Установка и запуск

### Базовая установка (без синхронизации)

Приложение работает полностью офлайн без дополнительной настройки.

### Настройка синхронизации (опционально)

Если вы хотите синхронизировать данные между устройствами:

1. Следуйте инструкциям в [SUPABASE_SETUP.md](./SUPABASE_SETUP.md)
2. Для получения API ключей см. [SUPABASE_API_KEYS.md](./SUPABASE_API_KEYS.md)
3. Создайте файл `.env.local` в корне проекта
4. Добавьте переменные окружения из Supabase

## Установка и запуск

### Требования

- Node.js версии 20.19+ или 22.12+
- npm или yarn

### Шаги установки

1. Клонируйте репозиторий или скачайте файлы проекта

2. Установите зависимости:
```bash
npm install
```

3. Запустите сервер разработки:
```bash
npm run dev
```

4. Откройте браузер и перейдите по адресу:
```
http://localhost:5173
```

### Сборка для продакшена

Для создания оптимизированной версии приложения:

```bash
npm run build
```

Готовые файлы будут в папке `dist/`. Вы можете развернуть их на любом статическом хостинге.

Для предварительного просмотра собранной версии:

```bash
npm run preview
```

## Использование

### Добавление напитка

1. Нажмите кнопку **"+ Добавить напиток"**
2. Заполните форму:
   - Выберите тип напитка из списка
   - Укажите сорт кофе
   - Введите количество кофе, воды и молока (если используется)
   - Укажите время приготовления в секундах
   - Добавьте заметки об особенностях приготовления
   - Оцените результат от 1 до 5 звезд
3. Нажмите **"Добавить"**

### Просмотр деталей

- Кликните на любую карточку напитка в списке
- Откроется модальное окно с полной информацией

### Редактирование

1. Откройте детали напитка
2. Нажмите **"Редактировать"**
3. Внесите изменения и нажмите **"Сохранить"**

### Удаление

- Нажмите кнопку **✕** на карточке напитка
- Подтвердите удаление

## Хранение данных

Все данные хранятся локально в браузере с использованием LocalStorage API под ключом `coffeeTrackData`. 

**Важно**: 
- Данные привязаны к конкретному браузеру и устройству
- При очистке данных браузера все записи будут удалены
- Для резервного копирования данных можно экспортировать их через Developer Tools браузера
